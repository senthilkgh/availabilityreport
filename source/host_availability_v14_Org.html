How it works: Run the script below from any Windows machine that can reach your servers. It reads your Dynatrace CSV, pings each server, and saves ping_results.csv — upload it in the Upload tab.

Save as ping_servers.ps1 and run as Administrator

# ping_servers.ps1 — reads DT CSV, pings each server, outputs ping_results.csv param( [string]$In = "dt_servers.csv", [string]$Out = "ping_results.csv", [int] $Timeout = 1000 ) $data = Import-Csv $In $col = ($data[0].PSObject.Properties.Name) | Where-Object { $_ -match 'server|host|name' } | Select-Object -First 1 $res = @() $i = 0 foreach ($r in $data) { $i++ $h = ($r.$col -split '[.\s]')[0].ToLower().Trim() Write-Progress -Activity "Pinging" -Status "$i / $($data.Count) : $h" -PercentComplete ($i / $data.Count * 100) $st = "Unreachable"; $ms = "" try { $ping = New-Object System.Net.NetworkInformation.Ping $reply = $ping.Send($h, $Timeout) if ($reply.Status -eq "Success") { $st = "Reachable"; $ms = $reply.RoundtripTime } } catch { $st = "Error" } $res += [PSCustomObject]@{ ShortServerName = $h; OriginalName = $r.$col PingStatus = $st; LatencyMs = $ms CheckedAt = (Get-Date -Format "yyyy-MM-dd HH:mm") } } $res | Export-Csv $Out -NoTypeInformation -Encoding UTF8 Write-Host "Done: $Out" -ForegroundColor Green

First run: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
Then: .\ping_servers.ps1

How it works: Run the playbook from your jump server using a privileged user with become: yes / become_user: root. Supply the server list via -i inventory.ini. The playbook checks boot logs, writes per-server CSVs, combines them into ansible_check_combined.csv, and emails it using the system mail command.

Go to Review & Categorize tab → filter by "-- Check Manually rows --" → click ⬇ Download Hostnames for Ansible to get check_manually_hosts.txt.
